# PBKK-SpringMVC
Tugas Spring MVC - Form & Model Binding

## Deskripsi Aplikasi

```
Aplikasi menampilkan hasil inputan dari sebuah form.
Inputan aplikasi terdiri dari:
- Car Name
- Car Description
- Car Rarity
- Ability Chance
- Ability Value
```
